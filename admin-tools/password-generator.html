Password Hash Generator

Generate SHA-256 hashes for secure admin passwords

Security Note: Use this tool locally only. Never generate passwords on untrusted websites.

How to Use

  1. Enter a strong password above (at least 12 characters recommended)
  2. Click "Generate Hash" to create the SHA-256 hash
  3. Copy the hash
  4. Open your index.html file
  5. Find the CONFIG section at the top
  6. Replace the adminPasswordHash value with your new hash

adminPasswordHash: "your-hash-here",

Password Requirements

  • At least 12 characters
  • Mix of uppercase and lowercase letters
  • At least one number
  • At least one special character (!@#$%^&*)
  • Not a dictionary word

This tool runs entirely in your browser. No data is sent anywhere.
